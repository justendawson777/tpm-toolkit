## Portfolio Projects Index

This repository contains three specialized sub-projects designed to solve systemic enterprise delivery friction:

### 1. Risks Analyzer (`/Risks Analyzer`)
* **The Problem:** Cross-functional dependencies are buried in silos, leading to reactive, late-stage delivery failures.
* **The Solution:** A Python and Pandas data engine that programmatically parses structured Jira issue schemas. It maps complex workstreams, runs graph-traversal logic to identify multi-hop cascading blocker nodes, and automatically flags high-probability timeline slippages.
* **TPM Focus:** Proactive Risk Mitigation, Algorithmic Critical-Path Modeling, Agile Governance.

### 2. PRD to Jira Converter (`/PRD to Jira`)
* **The Problem:** Translating a Product Requirement Document (PRD) into a clean Jira backlog requires hours of manual ticket creation, often leading to missing requirements or misaligned hierarchies.
* **The Solution:** An automation engine that parses technical product text or structured spec documents, maps the scope directly into the enterprise Agile framework hierarchy (Initiative > Epic > Story), and prepares a seamless, programmatic bulk-import layout.
* **TPM Focus:** Product Vision Alignment, Automated Scoping, Process Streamlining.

### 3. Cloud Cost Auditor (`/Cloud Cost Auditor`)
* **The Problem:** Unoptimized enterprise cloud architectures lead to massive financial inefficiencies and unexpected infrastructure spending.
* **The Solution:** A compliance and resource tracking script that audits active cloud configurations, evaluates allocation footprints, and isolates orphaned assets to drive structural overhead optimization.
* **TPM Focus:** Engineering Cost-Reduction, Resource Optimization, Cloud Infrastructure Management.
